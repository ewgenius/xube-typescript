## Для запуска

### 1. Зависимости проекта
```
#!bash
sudo npm install typescript -g
sudo npm install bower -g
sudo npm install grunt-cli -g
npm install
bower install

```

### 2. Компиляция скриптов
```
#!bash
grunt

```

### 3. Запуск
```
#!bash
python -m SimpleHTTPServer

```
